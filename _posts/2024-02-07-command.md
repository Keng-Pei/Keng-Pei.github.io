# 清單  
要在段落裡面再加一個段落要先加一個TAB
## 有順序清單
1. 第1段
    1. 第1.1段
    2. 第1.2段
2. 第2段
3. 第3段
    1. 第3.1段

## 無順序清單
* 第1段
* 第2段
    * 第2.1段
    * 第2.2段
* 第3段    
## 待辦清單
-  [ ] 牛奶
-  [ ] 啤酒
-  [ ] 紅茶
-  [x] 綠茶

```md
# 清單  
## 有順序清單
1. 第1段
    1. 第1.1段
    2. 第1.2段
2. 第2段
3. 第3段
    1. 第3.1段

## 無順序清單
* 第1段
* 第2段
    * 第2.1段
    * 第2.2段
* 第3段    
## 待辦清單
-  [ ] 牛奶
-  [ ] 啤酒
-  [ ] 紅茶
-  [x] 綠茶
```
